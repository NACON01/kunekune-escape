# クネクネ離脱システム

スマートフォンで動画を見ながら歩く人を、目的地までの経路に沿ってクネクネ誘導するプロトタイプです。

Phase 1 の擬似モードに加えて、Phase 2a では端末の方位センサーと歩数検出を使うセンサーモードを実装しています。

## 起動

```bash
npm install
npm run dev -- --host
```

表示された URL を、同じ Wi-Fi に接続したスマートフォンから開いてください。センサーは HTTPS または localhost で利用してください。

## 使い方

### 擬似モード

「擬似モード」を選ぶと、従来どおりデバッグパネルの誘導方向・残り距離スライダーで動作を確認できます。

### センサーモード

1. 「センサーモード」を選び、「センサーを有効化」をタップします。
2. クネクネ地点で「経路の記録開始」をタップします。
3. 実際に目的地まで歩き、歩数と方位を記録します。デバッグパネルには現在方位・累計歩数・区間数が常時表示されます。
4. 目的地で「ここが目的地」をタップすると、経路がブラウザの localStorage に保存されます。
5. 「クネクネ離脱 開始」をタップすると、記録した区間を先頭から誘導します。保存済みの経路は「前回の経路で開始」から再利用できます。

### iPhone の権限許可

1. iOS 13 以降では、Safari で HTTPS の URL を開きます。GitHub Pages または HTTPS 化した開発環境を使ってください。
2. センサーモードで「センサーを有効化」を、端末を実際に使う向きでタップします。
3. 表示された「モーションと方向」へのアクセス確認で「許可」を選びます。方向とモーションの両方を許可してください。
4. 拒否した場合は、iPhone の「設定」→「Safari」→「モーションと方向のアクセス」を確認し、ページを再読み込みしてもう一度ボタンをタップします。

歩数検出は端末の加速度から概算しています。ポケットや手持ちなど端末の持ち方、床の振動によって検出精度が変わります。
